Current Problem

	•	Create or update the driver record in the drivers table
	•	Ensure it is correctly linked to:
	•	auth.users.id

	•	The drivers table must contain:

	•	User reference

	•	Push notification token(s)
	•	Any fields used in job assignment queries

2. Push Notifications – MUST Work When App Is Closed

Current Problem
	•	Drivers only receive jobs when the app is open.
	•	This means push notifications are not functioning correctly.

Required Fix
	•	Ensure:
	•	Push tokens are captured and stored correctly in the drivers table
	•	Tokens are updated on login and app reinstall
	•	Notifications are sent using the stored token at assignment time
	•	Job assignment must not rely on polling or app refresh

Verification
	•	Assign a job when:
	•	App is open
	•	App is in background
	•	App is fully closed
	•	Driver must:
	•	Receive push notification
	•	Be able to open the job directly from notification

If this fails in any state, the issue is not fixed.
